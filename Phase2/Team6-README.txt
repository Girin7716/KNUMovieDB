Oracle SQL 사용
1) DB_PHASE2.sql의 명령어를 사용하여 Table생성 및 관계 설정과 Data Set 삽입.
1-1)DATA SET 설명
Account.txt	: 고객 data set
Actor.txt		: 배우 data set 
Appear.txt		: 배우와 출연한 영화의 relationship을 나타낸 data set
Episode.txt	: Tv_series의 영화 중 episode의 시즌 정보와 회차 정보를 가지는 data set
Genre.txt		: 장르 종류의 data set
Movie.txt		: 영상물 300개 data set
Movieis.txt	: 영상물이 어떤 장르인지에 대한 relationship을 나타낸 data set
Rating.txt		: 고객이 영화에 점수를 줬는지에 대한 relationship을 나타낸 data set
Version.txt	: 각 영상물의 Version을 나타내는 data set
